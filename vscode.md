
# vscode快捷键
打开默认终端: ctrl+[`]   
新建一个终端：ctrl+shift+[`]  


# vscode中git

## remote命令
第一次使用的时候要手动绑定一个自己的远程GitHub库：  
```
git remote add origin
eg：
git remote add origin https://github.com/kitefano/huashan
```
然后push，这时候远程仓库里就更新了。  
之后就可以直接push了。  
如果要push别的仓库，则需要重新remote。

## 语法
### 行，段
行： 结尾两个空格，另起一行。  
段： 结尾空两格， 另起一行，再空一行。
